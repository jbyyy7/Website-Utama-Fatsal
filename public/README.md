# Public Assets

This directory contains static assets that will be served at the root of your domain.

## Usage

Place images, fonts, and other static files here:

```
public/
├── images/
│   ├── logo.png
│   └── hero-bg.jpg
├── favicon.ico
└── robots.txt
```

Files in this directory will be accessible at:
- `/images/logo.png`
- `/favicon.ico`
- etc.
